# Diplomacy Mobile App

Mobile application for the classic board game Diplomacy.

## Repository structure

```
├── backend/        # Kotlin/Spring Boot API server (Gradle)
├── ios/            # iOS client (Xcode/SwiftUI)
├── adjudication/   # Standalone Diplomacy adjudication engine (Kotlin/Gradle)
├── infra/          # Infrastructure as code (Terraform / CDK)
├── docs/           # Project documentation
└── Context/        # Product specs & design docs
```

## Project context

Specifications and design docs live in the **Context** folder:

- **Diplomacy_Mobile_App_PRD.docx** – Product requirements
- **Diplomacy_API_Specification.docx** – API spec
- **Diplomacy_Backend_Architecture.docx** – Backend architecture
- **Diplomacy_Data_Models_Schema.docx** – Data models & schema
- **Diplomacy_UIUX_Specification.docx** – UI/UX spec
- **Adjudication_Engine_Technical_Spec.docx** – Adjudication engine
- **Development_Plan.docx** – Development plan
